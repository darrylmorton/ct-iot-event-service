# ct-iot-event-service

## Requirements
- Go >= v1.20
- Docker Desktop >= v4.17.0 (99724)

## Build
```
make build
```

## Docker Compose Startup
```
docker compose up
```

## Test
```
make test
```

## Docker Compose Shutdown
```
docker compose down -v
```
